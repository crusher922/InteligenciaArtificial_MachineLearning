# Deber 1 — Modelos de Aprendizaje Automático

Este proyecto implementa distintos modelos de **Machine Learning** utilizando Python y la biblioteca **scikit-learn**.  
El objetivo es aplicar modelos supervisados a diferentes problemas prácticos y analizar sus resultados.

Los modelos implementados son:

1. Regresión Lineal — Predicción de consumo eléctrico
2. Árbol de Decisión — Aprobación de crédito
3. Random Forest — Detección de fraude en tarjetas
4. XGBoost — Predicción de riesgo crediticio
5. Support Vector Machine (SVM) — Clasificación de imágenes (perros vs gatos)
